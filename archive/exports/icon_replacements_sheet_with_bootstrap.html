Как использовать эту колонку

  • Если собственная иконка нравится больше — оставляешь свой SVG как primary.
  • Если нужно быстро проверить альтернативу — открываешь ссылку на Bootstrap и сравниваешь форму, stroke и читаемость.
  • Для control icons Bootstrap часто даёт хорошие готовые варианты: x-lg, arrow-left, chevron-left, play-fill.
  • Для смысловых иконок лучше смотреть не буквальное совпадение emoji, а наиболее читаемый символ из каталога.

Где смотреть в проекте

  • index.html — статическая разметка hero, safety, FAQ groups, CTA, popups.
  • src/scripts/main.js — runtime вставки стрелок, play, lightbox controls, copied/success states.
  • src/styles/main.css — базовое выравнивание через .ac-icon.
Для части emoji нет буквального 1:1 аналога в Bootstrap Icons, поэтому здесь даны подходящие варианты по смыслу. Например, для 🤖 логичен cpu-fill, а для 🏊 — life-preserver или water.
