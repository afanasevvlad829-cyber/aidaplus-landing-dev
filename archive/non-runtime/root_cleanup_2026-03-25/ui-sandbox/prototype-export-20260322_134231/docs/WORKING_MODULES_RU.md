# Рабочие модули сайта (без служебного audit-режима)

## 1) Верхнее меню и режимы
- Верхнее меню переключает контентные вкладки.
- Переключатель режима (подробно) меняет отображение модулей без перезагрузки.
- Кнопка "Связаться" открывает выпадающее меню контактов.

## 2) Выбор возраста
- Глобальный выбор возраста: 7-9 / 10-12 / 13-14.
- Выбор влияет на карточки смен и контентные блоки.
- Возраст сохраняется в localStorage.

## 3) Воронка и смены
- Окно смен открывается по CTA в Hero Right.
- В карточке смены:
  - "Показать мою цену" запускает первый шаг подбора цены.
  - "Улучшить цену ещё раз" запускает второй шаг (если доступен).
  - "Забронировать место" переносит в блок заявки.
- Статус выбранной смены и промокода сохраняется в localStorage.

## 4) Блок заявки (Hero Right)
- Форма: телефон, имя, согласие на ПДн, кнопка отправки.
- Без галочки согласия отправка блокируется.
- После отправки показывается подтверждение.
- Данные заявки могут отправляться в Telegram при настроенном AC_NOTIFY_CONFIG.

## 5) Моталки (карусели)
- Фото: карточки с фильтрами, открытие в попапе, навигация стрелками.
- Видео: карточки видео, переключение стрелками, просмотр в попапе.
- Отзывы: карусель карточек, переключение стрелками.
- Команда: карусель карточек, переключение стрелками.

## 6) Всплывающие окна (popups/overlays)
- Смены и цены.
- Галерея фото.
- Видео.
- FAQ.
- Подтверждение/статус заявки.

## 7) Что исключено из этого пакета
- Служебные audit-метки, панели и режимы разработки.
- Вспомогательные служебные инструменты разметки.
