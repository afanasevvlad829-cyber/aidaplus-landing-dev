# Task 1 — Full mode container layout fix

## Проблема
В full mode часть секций растягивается на всю ширину экрана.
Особенно это видно в секции AI-программы.
Нужно, чтобы full-mode секции были выровнены так же, как:
- Программа по возрастам
- Формат и условия

## Проверить секции
- #ai
- #location
- #photos
- #video
- #reviews
- #team
- #faq

## Что нужно сделать
- убедиться, что у каждой секции весь контент находится внутри центрирующего inner-container
- использовать тот же container/layout pattern, что у секций:
  - Программа по возрастам
  - Формат и условия
- ограничить ширину не только section head, но и всего body/content секции
- исправить HTML structure и CSS только там, где это нужно

## Ожидаемый результат
Карточки, статистика, текст и медиа внутри full-mode секций:
- не растягиваются на всю ширину экрана
- аккуратно центрированы
- визуально соответствуют верхним аккуратным секциям

## Не менять
- booking logic
- compact mode
- hero business logic
- data sources

## Формат работы
- минимальный diff
- после выполнения обновить:
  - /ai/report.md
  - /ai/checklist.md
- в чат дать короткий отчёт на русском
