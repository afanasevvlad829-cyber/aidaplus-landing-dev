# AidaCamp icon pack

Contents:
- assets/icons/*.svg — separate SVG files (better for Codex than sprite)
- docs/icon-system.css — base CSS for unified rendering
- docs/codex_replacement_map.md — exact map of where and what to replace
- docs/codex_task.txt — ready-to-paste Codex task

Recommended pattern:
<img class="ac-icon" src="/assets/icons/close.svg" alt="" aria-hidden="true">

This pack is intentionally organized to avoid rescanning the whole site.
