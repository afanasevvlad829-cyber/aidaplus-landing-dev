# Responsive Contract

## Viewports
- Desktop: 1440
- Tablet: 768
- Mobile: 390

## Expectations
- No horizontal overflow for product UI.
- Toggle remains visible in compact/full.
- Funnel controls remain usable.
- Overlays are reachable and closable.

## Wrapping Rules
- Full top navigation uses no-wrap behavior and horizontal overflow when needed.
- Compact menu keeps icon chips accessible.
